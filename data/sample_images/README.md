# Sample Images

This directory should contain test images for benchmarking.

## Required Files

- `test.jpg` - A sample image used for performance benchmarking

## Usage

Add your own test images here or download sample images from:

- COCO dataset: <https://cocodataset.org/>
- ImageNet: <https://www.image-net.org/>
- Or use your own photos

The benchmark feature in the app requires at least one `test.jpg` file in this directory.
